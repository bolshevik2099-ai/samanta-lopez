# Procesa-T CRM

CRM de Transporte para Samanta López.

Este proyecto es una interfaz avanzada para la gestión logística, conectada con AppSheet API para el registro de gastos, control de viajes y administración de flota.

## Estructura del Proyecto
- `login.html`: Acceso centralizado por roles.
- `vista-chofer.html`: Registro de gastos móvil-first.
- `vista-admin.html`: Panel de control administrativo.
- `vista-superadmin.html`: Gestión total del sistema.
- `js/`: Lógica de autenticación y conexión API.
